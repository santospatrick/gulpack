Ultimate Toolkit

This is a Toolkit but also a Boilerplate to use Gulp & Webpack We consider to use both technologies because projects need to handle images, fonts, css & javascript optimizations. This boilerplate also uses Browser-sync, Postcss &

Getting Started

  1. Install node.js
  2. "npm install --global gulp webpack" in your command line
  3. "npm install" in your command line to install packages
  4. "gulp" in your command line to start watching your "/source" file changes!
  5. Enjoy it!

Production Build

    Run npm run build to deploy styleguide, minified css & javascript
